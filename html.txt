Atributo HTML

São palavras especiais utilizadas para "configurar"/modificar/ uma tag html.

Esses atributos HTML são escritos dentro das tags de abertura.

eles tem uma aparência semelhante:

palavra=""

Uma palavra, um sinal de igual e duas áspas duplas.






Existem duas classficações de atributos HTML:

Atributos Globais


São atributos que podemos utilizar em qualquer tag HTML.



name=""







Atributos específicos para determinadas tags HTML


São atributos que só funcionam ou só tem sentido em determinadas tags HTML.


charset=""

caractere / colocar


Esse atributo é específico da tag HTML <meta>


metadados => São informações por trás de algum arquivo.
